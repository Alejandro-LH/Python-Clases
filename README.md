# Apuntes y Ejercicios de Python

En este repositorio comparto mis apuntes y ejercicios de clase en formato Jupyter Notebook. Aquí voy reuniendo los temas que he visto sobre introducción a Python, junto con ejemplos y pruebas sencillas que me ayudan a entender mejor cada concepto.

La idea de este material es mostrar, desde mi proceso de aprendizaje, cómo he ido trabajando la lógica de programación, la sintaxis básica del lenguaje y varios ejercicios prácticos vistos en clase.

## Contenido principal

El repositorio está centrado en el notebook [Clases.ipynb](/Users/alejandrolh/Downloads/Clases.ipynb), donde se trabajan temas como:

- criterios de evaluación y notas generales del curso
- paradigmas de programación
- tipos de datos en Python
- conversión de tipos y uso de anotaciones
- operadores aritméticos y lógicos
- condicionales
- funciones
- listas, tuplas, `range`, conjuntos y diccionarios
- ejemplos básicos de sintaxis y estructuras del lenguaje

## Objetivo del repositorio

Este repositorio sirve como apoyo de estudio para repasar los conceptos vistos en clase y practicar con ejemplos sencillos en Python.

Puede ser útil para:

- estudiantes que estén comenzando con Python
- repasar sintaxis básica
- consultar ejemplos rápidos de estructuras y funciones
- practicar lógica de programación
